# IoT-RGB-matrix

## Functionalities

- Sending custom images (and animations) over WiFi to the display.
- Automatically generate a QR code for display (one application is to rickroll people)
- Serving as a "status" display like date/time, weather, etc.

## Project Components

- 40x40 RGB LED matrix, which is broken down into 25 PCBs of 8x8 matrix
- A control board, which is likely a Raspberry Pi Zero W
